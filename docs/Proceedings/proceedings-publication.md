# Publishing final proceedings on JACoW.org

The Editor-in-Chief is responsible for creating the final proceedings, do all checks to ensure metadata correctness (authors list, titles, etc.) and assure that all speakers have given permission to eventually publish their slides.

When these requirements are fulfilled, the proceedings package needs to be delivered to the JACoW Proceedings Webmaster.

Basically the procedure is as follow:

1. the Editor-in-Chief notifies the JACoW Proceedings Webmaster (address available at [JACoW About / Contacts](https://www.jacow.org/About/BoardContacts)) webpage

2. the JACoW Proceedings Webmaster double checks the structure of the proceedings, looking for technical issues (missing parts, broken links, etc.)

3. if any issues are found, the Editor-in-Chief is asked to resolve them and re-create the proceedings

4. when everything's look fine, the JACoW Proceedings Webmaster will
   
     1. [register the DOI's](../doi)
   
     2. publish the proceedings package on [www.JACoW.org](https://proceedings.JACoW.org)
   
     3. notify the Editor-in-Chief about the success of the operation
